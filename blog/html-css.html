INLINE & INLINE-BLOCK

Overview

The CSS display property specifies the box type used for HTML elements. Elements with inline and inline-block display are both able to be placed within lines with adjacent content as they do not have default line-breaks placed afterwards. This is the property that sets them apart from block display elements which cannot be placed beside other content and take up the full line.

Property display: inline display: inline-block
Ability to set width and height No Yes
Width Minimum required Set width
Height Minimum required Set height
Left/right margins/padding Will not override Will override
Top/bottom margins/padding Will override Will not override
Box breaking Yes No
